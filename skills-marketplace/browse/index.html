How Buying Works

Built Around Archie

Archie is our desktop app — the same OpenClaw-powered assistant you already know, minus needing VS Code to run it. A proper interface for starting conversations, reviewing what your assistant did, and managing skills.

Buy a Skill Once

  • Browse and buy from this site or from inside Archie
  • Your account keeps track of everything you own
  • Skills sync automatically — no reinstalling per device
  • New skill versions update in place

Use It Everywhere

  • Works the moment you message your assistant, on any channel you've connected
  • No extra configuration beyond the initial "Add to my assistant"
