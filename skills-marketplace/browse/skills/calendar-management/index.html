Calendar Management

by Archie (built-in) v1.0.0 Free

Give your agent the ability to fully manage a Google Calendar. It can check availability, create events, reschedule meetings, set reminders, find free slots, and summarize upcoming schedules. All changes require confirmation before executing. Requires a connected Google account with calendar access.

Required Integrations

Google Calendar

Trigger Keywords

calendar schedule meeting appointment event book reschedule cancel event free time availability busy upcoming block time reminder agenda

Setup Steps

  1. Connect your Google account by clicking 'Connect Google' in the agent settings.
  2. Fill in your timezone (e.g. America/New_York).
  3. Set your working hours so the agent respects your schedule boundaries.
  4. Optionally add scheduling preferences (e.g. 'prefer mornings for focus time').

Configurable Variables

Variable Label Help
TIMEZONE Your timezone IANA timezone (e.g. America/Los_Angeles, Europe/London, Asia/Tokyo)
CALENDAR_ID Calendar ID Usually 'primary' for your main calendar. For a shared calendar, use its email address.
WORKING_HOURS_START Work day starts at 24-hour format (e.g. 09:00). The agent won't schedule events before this time unless you explicitly ask.
WORKING_HOURS_END Work day ends at 24-hour format (e.g. 17:00). The agent won't schedule events after this time unless you explicitly ask.
DEFAULT_EVENT_DURATION Default event duration (minutes) When no duration is specified, events will be created with this length.
SCHEDULING_PREFERENCES Scheduling preferences Any personal preferences for how events should be scheduled.
