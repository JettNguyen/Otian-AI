Submissions Open With Archie’s Launch For Developers

Build an Add-on, List It on the Marketplace

The marketplace is decentralized: anyone who can build an add-on can list one. We handle distribution and syncing — you focus on the add-on itself. Register interest and we’ll tell you when submissions open.

Getting Started

From Idea to Listed

Three steps from idea to listed.

01

Build Your Add-on

Package what you want to add to an agent — a skill (a new capability), a worker, a routine, or a personality. We'll publish the full developer spec soon.

Submit for Review

Submit your add-on for review. We check that it works as described and is safe to run before it goes live.

02
03

Go Live

Your add-on goes live in the marketplace, available from the website and from inside Archie.

Submit

Submit an Add-on

Pick what you built — each kind has its own short form. Describe it, paste its template, attach a screenshot of it working, and we'll review it. You'll sign in with your Archie account, since an add-on has to be built and tested in the app first.

Skill

A capability your agent gains — a connection to an app, a workflow, a specific task it can carry out.

Submit a skill →

Worker

A specialist your agent can hand a job to — a researcher, a writer, an expert for one kind of task.

Submit a worker →

Routine

Something that runs on a schedule — a morning briefing, a weekly summary, a recurring check-in.

Submit a routine →

Personality

A voice and manner for the agent — how it writes, how formal it is, the tone it strikes.

Submit a personality →

Not sure which one fits? Start here and we'll point you to the right form.
